Main properties to customize your text

You can personalize the color, font, size, and many other properties of your text

    Main properties to customize your text:
  • color: Changes the color of the text
  • font-family: Changes the font of the text
  • font-size: Changes the size of the font of the text
  • font-weight: Changes the thickness of the characters of the text
  • font-style: Changes the way the text appears on the screen (italic, normal, oblique, inherit, initial, unset)
  • font-height: Changes the space between lines of text
  • text-decoration: Adds a decoration to the text (underline, line-through)
  • text-align: Aligns the text (justify, center, right, left)
  • text-transform: Manages upper-lower case characters (lowecase, uppercase, capitalize, initial)
This is an example of what you can do to personalize your text
.text_customization {
background-color: yellow;
color: black;
font-family: Georgia;
font-size: 30px;
font-weight: 300;
font-style: oblique;
text-decoration: dotted;
text-align: center;
text-transform: capitalize;
}
