CSS box model

HTML has a format in which to organize the elements on a web page. You must have this in mind to insert new elements into it. There are two types of elements that follow this format:

Block-level items

These elements start on a new line every time you include them into the html code:

  • Headings
  • Paragraphs
  • Sections, articles, divs
  • ...

Inline items

These elements sit surronding the existing content, usually on the same line

  • Images, audio, video
  • Spans
  • ...

Element selectors

CSS must refer to the elements in your html document to apply their styles. These are the selectors you can use to point to elements (in order of preference)

  • Inline styles as an attribute

    • ... style="color:green; font-family: 'Times New Roman'"; ...

  • Using the ID of the element

    • #using_id_to_style {
    color: brown;
    }

    ... id="using_id_to_style" ...

  • Using a class

    • .list_element_class {
    background-color: blue;
    color: white;
    font-family: Verdana, Geneva, Tahoma, sans-serif;
    }

    ... class="list_element_class" ...


    • [type='submit'] {
    background-color: darkblue;
    color: darkorange;
    }

    ... type="submit" ...

  • !important
    overrides every other style by any given selector when placed next to a certain property

CSS length units

CSS allows to change the size and properties that involve measures of elements. There are two types of length units on which you can insert these measures:

  • Absolute: They tie to physical units of length (e.g. in, mm or px)
  • Relative: They are relative to another length value (e.g. em or rem)

Colors

The colors you put and the way you put them in your web page must convey information about what you're saying to the final user. The following sections will provide you with the basics of color theory and how to represent colors in CSS

Color theory

Basic colors

Digital screens can display these three basic colors:

  • Red
  • Green
  • Blue

The combination of them is what we perceive as the other colors of the spectrum, as we can see up next

Secondary colors

These are the result of the combination of two basic colors

  • Yellow
  • Cyan
  • Magenta

Tertiary colors

These are the combination of a basic and a secondary color

  • Orange
  • Raspberry

Complementary colors

  • Blue & Yellow
  • Green & Magenta
  • Red & Cyan

CSS color representation

In CSS, you can represent colors in the following ways:

  • Plain words
    (e.g. black , white , blue )
  • Hexagesimal values
    (e.g. #000000 , #FFF , #00F )
  • RGB notation
    (e.g. rgb(0,0,0) , rgb(255,255,255) , rgb(0,0,255) )
  • RGBA notation
    (e.g. rgba(0,0,0,1) , rgba(255,255,255,1) , rgba(0,0,255,1) )
  • Using hsl(hue, %saturation, %lightness) property
    (e.g. hsl(0,0%,0%) , hsl(360,0%,100%) , hsl(240,100%,50%) )

CSS variables

They allow you to change various properties at the same time

This is a sample text to show you what you can do with CSS variables

.using_css_variables {
background-color: var(--var_color, lightgreen);
font-weight: var(--var_weight, light);

... class="using_css_variables" style="--var_color: yellow; --var_weight: bold;" ... }

CSS pseudo-classes

A pseudo-class is a class that is applied to the element it's referring to when certain event happens.

Some pseudo-classes:
  • root: Is the default class that is applied to the web page if any other styles override it
  • hover: Event that fires when the mouse hovers over the element
  • focus: Event that fires when the user focuses over an element (generally using TAB key)

Example:

Hover over me!

.pseudo-classes_example {
color: white;
background-color: rgba(0, 0, 255, 0.6);
font-family: Lobster;
font-size: 30px;
}
.pseudo-classes_example:hover {
color: black;
background-color: rgba(255,255,0,1);
}

CSS pseudo-elements

A pseudo-element is a new element that doesn't belong to the html document (it's inserted purely by CSS)

Example:


.heart {
position: relative;
margin: 20px;
top: 0;
right: 0;
bottom: 0;
left: 0;
background-color: magenta;
height: 50px;
width: 50px;
transform: rotate(-45deg);
}
.heart::after {
background-color: magenta;
content: "";
border-radius: 50%;
position: absolute;
width: 50px;
height: 50px;
top: 0px;
left: 25px;
}
.heart::before {
content: "";
background-color: magenta;
border-radius: 50%;
position: absolute;
width: 50px;
height: 50px;
top: -25px;
left: 0px;
}
... class="heart" ...

Some pseudo-elements:
  • before: Used to add something before a selected element
  • after: Used to add something after a selected element
