The position property

This property defines the way the element is positioned on the web page

    position can have the following values:
  • static: Is the default position value that an element has when it's inserted into the flow. It cannot be offsetted
  • relative: Keeps the element on the normal flow of the page, unlocking offsetting
  • absolute: Takes out the element of the normal flow and places it relative to its parent container
  • fixed: Locks the element relative to the browser window. The element doesn't move when the user scrolls! It doesn't interfere with the position of other elements.
  • sticky: At first, the element behaves like a relative positioned element. When the user scrolls the wheel, it sticks to the top or bottom of the window until it reaches the end of its parent element

You can further modify the position of the elements with top, right, bottom, or left properties

    NOTES:
  • A sticky and fixed positioned elements must have top or bottom properties to function properly.
  • An element with an absolute position, has to be inside another relative or absolute element to be positioned properly.

The margin property

This property defines how much space there is between the element and its surroundings.

    margin can have the following values:
  • (numeric value)
  • auto: Positions the element at the center of its parent element.

You can individually adjust each side's margin with margin-top, margin-right, margin-bottom, or margin-left.

The padding property

This property allows to change how much space there is between the element's content and element's border

padding property can only take numerical values

Advanced properties

The following properties will allow you to further custom the position of your elements:

float

This property removes the element from the normal flow of the document and pushes it either left or right of their containing parent element
